linux基础操作+终端科学上网

2020-10-22

1.创建文件

touch <file>
touch {1..10}.txt

2.进入某个目录

cd /文件名/...

3.返回上一个目录

cd ..

4.查看当前所在目录的绝对路径

pwd

5.ls命令

ls [-alrAFR] [name...]

-a 显示所有文件及目录 (. 开头的隐藏文件不会列出)
-l 除文件名称外，亦将文件型态、权限、拥有者、文件大小等资讯详细列出
-r 将文件以相反次序显示(原定依英文字母次序)
-t 将文件依建立时间之先后次序列出
-A 同 -a ，但不列出 "." (目前目录) 及 ".." (父目录)
-F 在列出的文件名称后加一符号；例如可执行档则加 "*", 目录则加 "/"
-R 若目录下有文件，则以下之文件亦皆依序列出

6.更新源

sudo apt-get update
sudo apt-get upgrade

7.安装命令

sudo apt-get install <所要安装的名称>

二，用户及文件权限管理
1.查看用户(pts表示伪终端)

who am i
who mom likes

2.创建用户

sudo adduser <用户名>

3.切换用户

su <用户名> 或者 su -l <用户名>

4.用户组

groups <用户名>

5.查看/etc/group文件

cat /etc/group | sort         # | sort 表示将读取到的文本进行一个字典排序再输出
cat /etc/group | grep -E "<用户名>"     # | grep -E 过滤其他信息，只看引号关键字的

6.将其他用户假如sudo用户组(使其用户能够sudo)

sudo usermod -G sudo <用户名>

7.删除用户和用户组

sudo deluser <用户名> --remove-home   #后面的--remove-home不能去掉，表示删除user时一并将/home下的对应用户文件夹删除

8.查看文件权限

ls -l            # 查看本目录下的文件权限
ls -l <文件名>      # 查看某文件的权限

三，文件目录
1.认识目录结构

2.返回你的用户home目录

cd ~

3.新建空白文件

touch <文件>

4.新建目录

mkdir <目录名> 或 mkdir -p <father>/<son>/<grandsom>   # 新建毕竟深目录必须加-p

5.复制文件

cp <源文件> <复制地点>      # 复制单个文件
cp -r <文件目录> <复制地点>   #复制目录到某个地点，-r表示递归复制

6.删除文件

rm <文件名>   # 删除单个文件
rm -f <文件名>   # 忽略提示直接强制删除

7.删除目录

rm -r <目录名>
rm -rf <目录名>
